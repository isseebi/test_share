# Servo Parameter Tuning Assist Bot

このプロジェクトは、サーボモータの制御シミュレーションを行い、LLM（大規模言語モデル）のアシストを受けてパラメータ調整を効率化するためのツールです。

## 概要

`servo_lit.py`は、Streamlitを使用したGUIベースのシミュレーションツールです。1慣性系および2慣性系の物理モデルをシミュレートし、PIDゲインやフィルタ設定が制御挙動（位置、速度、トルクなど）に与える影響をリアルタイムに確認できます。さらに、AIチャット機能を通じて、現在の波形やパラメータに基づいた調整アドバイスを受けることができます。

## 主な機能

- **物理シミュレーション**:
  - **1慣性系 (Single Inertia)**: シンプルな回転体モデル。
  - **2慣性系 (Two Inertia)**: モータと負荷が軸剛性で結合された、より実機に近い振動成分を含むモデル。
- **PID制御 & フィルタ**:
  - 位置ループのPID制御、出力制限（トルクリミット）の設定。
  - 指令LPF（ローパスフィルタ）および出力LPFの適用。
- **解析・可視化**:
  - ステップ応答の波形表示（全体図および拡大図）。
  - オーバーシュート、制定時間（±2%）の自動算出。
  - 前回実行時との比較（メトリクス表示）。
- **AI調整アシスト**:
  - LLLMを使用したチャットインターフェース。
  - シミュレーション波形（画像）をチャットに送信し、視覚的な挙動解析に基づいたアドバイスを取得。
  - 調整履歴の管理と文脈に沿った対話。

## セットアップ

### 1. 必要ライブラリのインストール

Python 3.11で動作確認済み。

```bash
pip install streamlit numpy matplotlib langchain
```

### 2. APIキーの設定

プロジェクトのルートディレクトリに `api_key.txt` を作成し、以下の形式でAPIキーを入力してください。

```text
API_KEY:your_gemini_api_key_here
```

## 使い方

以下のコマンドを実行すると、ブラウザでツールが立ち上がります。

```bash
streamlit run servo_lit.py
```

1. 左側のサイドバーから物理モデル（1慣性 / 2慣性）を選択します。
2. 慣性、摩擦、PIDゲインなどのパラメータを設定します。
3. 「シミュレーション実行 / 更新」をクリックして波形を確認します。
4. グラフの右上にある「➕」ボタンを押すと、その画像がAIへの相談用として添付されます。
5. 下部のチャット欄で「オーバーシュートを抑えるにはどうすればいい？」などの相談を投げかけてください。

## ファイル構成

- `servo_lit.py`: メインのStreamlitアプリケーション。UIと全体の制御を担当。
- `servo_sim.py`: シミュレーションのリポトリ。物理モデルの構築とシミュレーション実行のエントリポイント。
- `sample_sim_core.py`: PID制御器、LPF、モデルクラスなどのコアロジック。
- `LLM_call.py`: LLM (LangChain) との通信処理を担当。
- `analysis_drawing.py`: グラフ描画および統計量計算のユーティリティ。
- `PID_1inr_doc.md / PID_2inr_doc.md`: AIが参照する調整ノウハウのドキュメント。
