# システム概要
## プラント詳細
このコードでは、**ボールねじを用いた直線往復運動機構**の物理モデルがシミュレーションされています。
### 機構の特徴
* **システムの種類**: 1自由度の直線運動システム（1軸アクチュエータ）。
* **動力伝達**: サーボモータ等の入力を「力（Force）」として受け取り、可動部（テーブル等）を移動させる構成です。
* **物理的特性**:
    * **質量 (Mass)**: $1.5 \, \text{kg}$ の可動部。
    * **粘性摩擦 (Friction)**: 速度に比例する抵抗力（摩擦係数 $0.5$）が考慮されています。
### 運動方程式
$$m\frac{d^2x}{dt^2} = F - c\frac{dx}{dt}$$
（$m$: 質量, $x$: 位置, $F$: 入力, $c$: 粘性摩擦係数）
この方程式を**オイラー法**を用いて離散化し、加速度 → 速度 → 位置の順で計算することで、物体の挙動を再現しています。

## 制御則
このシステムでは、目標位置に精度よく停止させるための **PID制御（比例・積分・微分制御）** が採用されています。

### 制御方法
現在位置と目標位置の「偏差（誤差）」を解消するように、以下の3つの要素を組み合わせて操作量（力）を決定しています。
1. **P（比例）**: 偏差に比例して動かす。
2. **I（積分）**: 過去の偏差を累積し、定常偏差（目標とのわずかなズレ）を解消する。
3. **D（微分）**: 偏差の変化率を見て、急激な変化を抑える（ブレーキの役割）。


# パラメータ調整手順
## 前提条件
* 摩擦やイナーシャ（質量）などの物理モデルパラメータは固定とする。
* パラメータの調整は、比例ゲイン（$K_p$）、微分ゲイン（$K_d$）、積分ゲイン（$K_i$）の順に行う。

### Step 1：初期化と目標の確認
1. **パラメータの初期化**
   * すべてのゲイン（$K_p$, $K_i$, $K_d$）を `0` に設定する。
2. **評価基準（許容誤差）の設定**
   * 目標値に対して、どの程度の誤差範囲内に入れば「整定した」とみなすかを定義する（例：目標値の $\pm 1\%$ 以内など）。

### Step 2：比例ゲイン（$K_p$）の調整 ＝ 立ち上がり速度の確保
目的：目標値付近まで素早く移動させる（立ち上がり時間の短縮）。
1. $K_p$ を小さな値から徐々に大きくしていく。
2. システムの応答を確認し、目標値に到達するまでの時間が短くなることを確認する。
3. **ストップ条件**: 応答が目標値を通り過ぎる（オーバーシュートする）、あるいはシステムが振動し始めたら、$K_p$ の増加を止める。
   * *※この時点では、目標値に完全に一致しなくてもよい（定常偏差は後で消すため）。*

### Step 3：微分ゲイン（$K_d$）の調整 ＝ 振動とオーバーシュートの抑制
目的：ブレーキをかけて行き過ぎを防ぎ、ピタリと止める（整定時間の短縮）。
1. 第2段階で設定した $K_p$ を保持したまま、$K_d$ を小さな値から徐々に大きくしていく。
2. $K_p$ によって生じていた振動やオーバーシュートが収まる（減衰する）ことを確認する。
3. オーバーシュートが許容範囲内に収まり、最も早く目標値付近で安定する $K_d$ の値を探す。
4. **ストップ条件**: $K_d$ を大きくしすぎると、動きが過度に鈍くなり逆に到達時間が遅くなる、あるいは制御入力（力）が細かく振動する（ノイズの増幅）ため、その手前で止める。

### Step 4：積分ゲイン（$K_i$）の調整 ＝ 定常偏差（残差）の解消
目的：最後に残ったわずかなズレをなくし、正確に目標値に一致させる。
1. $K_p$ と $K_d$ を保持したまま、$K_i$ を非常に小さな値から徐々に増やしていく。
2. 目標値との間に残っていたズレ（定常偏差）がゼロに近づくことを確認する。
3. **ストップ条件**: $K_i$ を大きくしすぎると、再びオーバーシュートが発生し、結果的に「整定時間」が延びてしまう。ズレが実用上の許容時間内に解消される**必要最小限の値**にとどめる。

### Step 5：最終微調整（チューニング）
整定時間を極限まで短縮するためには、$K_p$ と $K_d$ のバランス（PD制御）が最も重要になる。
1. **さらに早くしたい場合**: $K_p$ を少し上げ、同時に増えた振動を抑えるために $K_d$ も少し上げる。これを振動が抑えきれなくなる限界まで繰り返す。
2. **$K_i$ の影響確認**: 最終的な挙動において、目標値付近でフワフワと波打つような動き（低周波の振動）がある場合は、$K_i$ が大きすぎるサインであるため、$K_i$ を下げる。

---

## 【重要】整定時間短縮のポイント
* 整定時間を短くする（素早くピタリと止める）ためには、**「$K_p$ をできるだけ大きくし、$K_d$ で強力にブレーキをかける」** アプローチが有効である。
* $K_i$ は整定時間を長引かせる最大の要因になりやすいため、**「必要最小限」** に設定することが基本となる。
